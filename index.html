Bot in Search of a Word

About:

This is a bot for the game Eight Letters in Search of a Word. It will read the player's screen and then entire all scoring words almost immediately. This bot simulates human input, albiet at superhuman speeds, meaning that its input is indistinguishable from that of a human player.

This bot is compatible with the games' singleplayer and multiplayer versions on all systems supporting the below requirements. Minor user customization may be required, and can be implemented using the settings.py file included in the download. All usage instructions can be found in the readme.txt included in the download.

Requirements:

  • Windows 7
  • Python 3.4+
  • PIL (Pillow is recommended)

Download:

The bot can be downloaded as Bot In Search of a Word.rar. This contains all necessary files for execution, excluding the required Python libraries. Be sure to read the included readme.txt.
